⚠️ This tool uses a pseudo-OTUI parser (jtoml-based), not full OTUI compatibility.

Known limitations

  • Only a simplified, jtoml-compatible pseudo-OTUI syntax is supported.
  • No full OTUI grammar support (e.g. templates, style inheritance, advanced declarations).
  • Only selected widgets are rendered: Label, Button, TextEdit, Panel, MiniWindow, UICreature, UIItem.
  • Only a limited subset of properties is interpreted (for example: size, margins, text).
  • Rendering uses plain HTML/CSS approximation and does not match full OTClient runtime behavior.
